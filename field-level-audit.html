Field Level Audit

Description

  • Client : HSBC

Field Level Audit application helps designer to implement audit at field level storing in a JSON format using generic rules designed for creating JSON audit trail using old and new value of a field and helps user to track updates to particular field values in an application.

Technology

  • Appian
  • mySQL

Role

Designed Generic Field Level Audit Module which allows to implement auditing the fields of any CDT in json format functionality and also allows to implement version comparison using audit logs.

Key Features

  • Auditing Flat CDTs
  • Auditing Fields with multiple values
  • Auditing Hierarchical Field.
  • Record version runtime Generation & Comparison
  • Storing Additional Information
  • Screen Capturing & Action Capturing
  • Single Row Audit for number of Fields
